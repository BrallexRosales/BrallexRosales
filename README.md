# BrallexRosales

A simple static website project structure containing:

- `index.html`
- `style.css`
- `script.js`
- `assets/`
  - `images/`
  - `videos/`
  - `music/`
  - `icons/`

## Usage

Open `index.html` in a browser to view the starter page.

Place your media assets in the appropriate folders under `assets/`.
